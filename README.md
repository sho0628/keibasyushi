# 競馬収支管理 (keibasyushi)

競馬の収支と回収率を管理するシンプルなWebアプリです。サーバー不要で、ブラウザのLocalStorageにデータを保存します。

## 機能

- **記録管理**: 日付・競馬場・レース番号・レース名・馬券種別・買い目・投資額・払戻金・メモを記録
- **サマリー表示**: 総投資額・総払戻金・収支・回収率・的中率を常時表示
- **集計**:
  - 月別 / 日別 (直近30日)
  - 馬券種別ごと (単勝・複勝・馬連・3連単 など)
  - 競馬場ごと
- **絞り込み**: テキスト検索 / 馬券種別 / 月による絞り込み
- **データ管理**: JSON / CSV エクスポート、JSON インポート、全データ削除

## 使い方

### ローカルで起動

サーバー不要です。`index.html` をブラウザで直接開くだけで動作します。

```bash
# Linux/macOS
xdg-open index.html  # or: open index.html

# Windows
start index.html
```

または、簡易HTTPサーバーを立てて開くこともできます。

```bash
python3 -m http.server 8000
# ブラウザで http://localhost:8000 を開く
```

### データ保存について

- データは利用ブラウザの **LocalStorage** に保存されます
- ブラウザのデータを消去するとデータも消えます
- 重要な記録は「データ管理」タブから定期的に **JSONエクスポート** してバックアップを取ることをおすすめします
